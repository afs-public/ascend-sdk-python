# State

The state of bank relationships to filter by. Unspecified returns relationships of all states.


## Values

| Name                | Value               |
| ------------------- | ------------------- |
| `STATE_UNSPECIFIED` | STATE_UNSPECIFIED   |
| `PENDING`           | PENDING             |
| `APPROVED`          | APPROVED            |
| `CANCELED`          | CANCELED            |
| `REJECTED`          | REJECTED            |
