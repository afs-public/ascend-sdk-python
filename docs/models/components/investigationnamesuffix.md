# InvestigationNameSuffix

The name suffix for individuals, one of:
- `NAME_SUFFIX_UNSPECIFIED` - Default/Null value.
- `SR` - Senior.
- `JR` - Junior.
- `III` - The third.
- `IV` - The fourth.
- `V` - The fifth.


## Values

| Name                      | Value                     |
| ------------------------- | ------------------------- |
| `NAME_SUFFIX_UNSPECIFIED` | NAME_SUFFIX_UNSPECIFIED   |
| `SR`                      | SR                        |
| `JR`                      | JR                        |
| `III`                     | III                       |
| `IV`                      | IV                        |
| `V`                       | V                         |
