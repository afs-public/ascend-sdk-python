# RedemptionFrequency

Indicates how frequently an account can redeem the asset.


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `FREQUENCY_UNSPECIFIED` | FREQUENCY_UNSPECIFIED   |
| `DAILY`                 | DAILY                   |
| `WEEKLY`                | WEEKLY                  |
| `BI_WEEKLY`             | BI_WEEKLY               |
| `MONTHLY`               | MONTHLY                 |
| `QUARTERLY`             | QUARTERLY               |
| `BI_ANNUALLY`           | BI_ANNUALLY             |
| `ANNUALLY`              | ANNUALLY                |
