# RecordState

Indicates the current state of the asset record.  The record state identifies which record types the asset allows.


## Values

| Name                       | Value                      |
| -------------------------- | -------------------------- |
| `RECORD_STATE_UNSPECIFIED` | RECORD_STATE_UNSPECIFIED   |
| `ACTIVE`                   | ACTIVE                     |
| `FROZEN`                   | FROZEN                     |
| `INACTIVE`                 | INACTIVE                   |
