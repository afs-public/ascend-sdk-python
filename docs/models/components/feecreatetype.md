# FeeCreateType

The type of fee being specified. Only the type of "BROKER_FEE" is supported.


## Values

| Name                   | Value                  |
| ---------------------- | ---------------------- |
| `FEE_TYPE_UNSPECIFIED` | FEE_TYPE_UNSPECIFIED   |
| `BROKER_FEE`           | BROKER_FEE             |
| `CONTRACT_FEE`         | CONTRACT_FEE           |
