# EntryConversionType

Corresponds to whether the event is CASH | STOCK | CASH_AND_STOCK


## Values

| Name                     | Value                    |
| ------------------------ | ------------------------ |
| `EVENT_TYPE_UNSPECIFIED` | EVENT_TYPE_UNSPECIFIED   |
| `CASH`                   | CASH                     |
| `STOCK`                  | STOCK                    |
| `CASH_AND_STOCK`         | CASH_AND_STOCK           |
