# OrderTradingSession

Which TradingSession to trade in, defaults to 'CORE'. Only available for Equity orders.


## Values

| Name                          | Value                         |
| ----------------------------- | ----------------------------- |
| `TRADING_SESSION_UNSPECIFIED` | TRADING_SESSION_UNSPECIFIED   |
| `CORE`                        | CORE                          |
| `PRE`                         | PRE                           |
| `POST`                        | POST                          |
| `OVERNIGHT`                   | OVERNIGHT                     |
| `APEX24`                      | APEX24                        |
| `GTX`                         | GTX                           |
