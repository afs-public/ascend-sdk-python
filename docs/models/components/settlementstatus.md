# SettlementStatus

Indicates the current status of the settlement process for the deliverable asset(s) tied to an option, such as regular settlement or delayed.


## Values

| Name                            | Value                           |
| ------------------------------- | ------------------------------- |
| `SETTLEMENT_STATUS_UNSPECIFIED` | SETTLEMENT_STATUS_UNSPECIFIED   |
| `REGULAR`                       | REGULAR                         |
| `DELAYED`                       | DELAYED                         |
