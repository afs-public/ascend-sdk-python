# OptionOrderCancelInitiator

Whether the cancel was initiated by the correspondent firm (FIRM) or the customer (CLIENT). Populated from the CancelOptionOrderRequest.


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `INITIATOR_UNSPECIFIED` | INITIATOR_UNSPECIFIED   |
| `FIRM`                  | FIRM                    |
| `CLIENT`                | CLIENT                  |
