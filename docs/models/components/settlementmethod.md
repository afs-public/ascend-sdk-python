# SettlementMethod

Specifies how an option is settled at expiration, either Physical (delivery of the underlying asset) or Cash (payment of the cash difference).


## Values

| Name                            | Value                           |
| ------------------------------- | ------------------------------- |
| `SETTLEMENT_METHOD_UNSPECIFIED` | SETTLEMENT_METHOD_UNSPECIFIED   |
| `CASH`                          | CASH                            |
| `PHYSICAL`                      | PHYSICAL                        |
