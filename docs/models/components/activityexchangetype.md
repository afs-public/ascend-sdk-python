# ActivityExchangeType

Indicates whether the merger was for cash, stock, or both


## Values

| Name                     | Value                    |
| ------------------------ | ------------------------ |
| `EVENT_TYPE_UNSPECIFIED` | EVENT_TYPE_UNSPECIFIED   |
| `CASH`                   | CASH                     |
| `STOCK`                  | STOCK                    |
| `CASH_AND_STOCK`         | CASH_AND_STOCK           |
