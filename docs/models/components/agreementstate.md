# AgreementState

The status of an agreement which blocks an enrollment; `REQUIRED` if not yet received, or `AFFIRMED` if acknowledgement has been received by AFS


## Values

| Name                          | Value                         |
| ----------------------------- | ----------------------------- |
| `AGREEMENT_STATE_UNSPECIFIED` | AGREEMENT_STATE_UNSPECIFIED   |
| `REQUIRED`                    | REQUIRED                      |
| `AFFIRMED`                    | AFFIRMED                      |
| `VOIDED`                      | VOIDED                        |
