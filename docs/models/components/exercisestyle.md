# ExerciseStyle

Indicates how and when an option can be exercised, either American (exercise any time before expiration) or European (exercise only at expiration).


## Values

| Name                         | Value                        |
| ---------------------------- | ---------------------------- |
| `EXERCISE_STYLE_UNSPECIFIED` | EXERCISE_STYLE_UNSPECIFIED   |
| `AMERICAN`                   | AMERICAN                     |
| `EUROPEAN`                   | EUROPEAN                     |
