# AccountState

Indicates if the account is `OPEN`, `PENDING`, or `CLOSED`


## Values

| Name                | Value               |
| ------------------- | ------------------- |
| `STATE_UNSPECIFIED` | STATE_UNSPECIFIED   |
| `PENDING`           | PENDING             |
| `OPEN`              | OPEN                |
| `CLOSED`            | CLOSED              |
