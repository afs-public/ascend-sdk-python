# SetExtraReportingDataResponseIdentifierType

The identifier type of the asset being ordered. For Equities: only SYMBOL is supported For Mutual Funds: only SYMBOL and CUSIP are supported


## Values

| Name     | Value    |
| -------- | -------- |
| `SYMBOL` | SYMBOL   |
| `CUSIP`  | CUSIP    |
| `ISIN`   | ISIN     |
